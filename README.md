# QueueSystem

This project control patients when it visiting the doctor it sort and handling the Queue so First patient come it well be the First one Meet doctor. 

## Project Tool

- Angular Cli v13.0.3
- Angular Matrial 
- Json-server as Fake API
- Angular Flex-Layout
